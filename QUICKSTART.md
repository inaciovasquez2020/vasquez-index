# Vasquez Index Quickstart

This is the shortest path from clone to a first successful local verification pass.

## Requirements

- `git`
- `bash`
- `python3`

## 1. Clone

```bash
git clone https://github.com/inaciovasquez2020/vasquez-index.git
cd vasquez-index
```

## 2. Check tools

```bash
python3 --version
git --version
```

## 3. Run the canonical repository check

```bash
make verify
```

## 4. Review the main navigation surfaces

- `README.md`
- `REGISTRY.md`
- `STATUS.md`
- `CANONICALITY.md`
- `FREEZE.md`
